# Customer Retention & Churn Analysis

## Project Overview
This project focuses on analyzing customer churn and retention behavior for a subscription-based business using Python. The objective is to identify why customers leave, understand retention patterns, and provide actionable business recommendations to improve customer loyalty and reduce churn.

This project was completed as part of the **Future Interns – Data Science & Analytics Task 2 (2026)** internship program.

---

# Objective
The main goals of this project are:

- Analyze customer churn patterns
- Identify key retention drivers
- Understand customer lifetime trends
- Measure customer retention behavior
- Generate business-focused insights
- Recommend strategies to reduce customer churn

---

# Dataset Used
Dataset: Telco Customer Churn Dataset

https://www.kaggle.com/datasets/blastchar/telco-customer-churn

The dataset contains:

# Project Workflow

## 1. Data Cleaning
- Removed missing values
- Converted data types

- Prepared dataset for analysis

## 2. Exploratory Data Analysis (EDA)
- Customer distribution analysis
- Churn distribution analysis

## 3. Churn Analysis
- Calculated overall churn rate

- Analyzed churn by:
  - Contract type
  - Payment method
  - Customer tenure

## 4. Retention Analysis
- Grouped customers by tenure
- Analyzed long-term customer behavior
- Studied customer retention trends

## 5. Customer Lifetime Value (CLV)
Calculated customer lifetime value using:

CLV = Monthly Charges × Customer Tenure

## 6. Business Insights & Recommendations
Generated actionable recommendations to:
- Reduce churn
- Improve retention
- Increase customer satisfaction
- Improve recurring revenue

---

# Key Insights

- Customers with month-to-month contracts showed the highest churn rate.
- Most customer churn occurred within the first 12 months.
- Customers using electronic check payment methods were more likely to churn.
- Long-term contract customers showed better retention.
- Higher monthly charges increased churn probability.
- Customers with higher tenure generated greater customer lifetime value (CLV).
